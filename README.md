大学時代、授業課題で自身の名前に因んだゲームを制作しました，  
コードは　game.pde　にあります．  
けんや→剣と矢を使った戦闘ゲーム．  
Processingで動作させることを想定しています．  
  
ルール  
・画面下の矢印ボタンで左右に移動可能．キーボードの矢印（←→）でも操作可能．  
・敵が近づいてくるので，敵の前に立つことで戦って勝つことが可能（＋15P)．  
・敵に陣地に入られれば減点（-5P)．  
・矢も飛んでくるので，当たってしまうと減点（-10P)．  
・画面右下にある矢のボタンで矢を飛ばして戦うことができる．キーボードのSPACEキーでも操作可能．  
・矢同士を衝突させれば加点（＋100P）．敵にあたっても加点（＋10P）．  
・矢を発射すると５秒間は次の矢を発射できない．残り秒数はインジケーターで確認可能．  

繰り返しプレイ可能．1回のプレイ30秒の制限時間つき．1回プレイが終わるごとにランキングに記録される．
